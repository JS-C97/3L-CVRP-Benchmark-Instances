# 3L-CVRP-Benchmark-Instances

A centralized directory of publicly available benchmark instances for the
three-dimensional loading capacitated vehicle routing problem (3L-CVRP) and its
variants.

This repository accompanies our survey paper and consolidates the benchmark
links discussed in it. For each dataset we searched the source originally cited
in the literature; where that source was no longer reachable, we searched for a
community-maintained mirror hosting the same instances. **Only datasets for
which a working link (original or mirror) could be confirmed are listed below.**
All links were manually checked. Please refer to the manuscript for a detailed
description of each dataset.

> **On data provenance.** Mirror links are provided solely to improve
> accessibility and reproducibility; the authoritative definition of each
> instance set remains the corresponding original publication, which users are
> encouraged to cite.
